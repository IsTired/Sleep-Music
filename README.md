Please if you see ANY problem make an issue, i will try to read through all of them, and if you have any tips please give me them, i need them :p

-discord.js v14
-distube
-libsodium
-libsodium-wrappers
-@distube/yt-dlp
-@distube/spotify
-@distube/soundcloud
-@discordjs/voice
-@discordjs/opus

I hope you enjoy this bot :)
